Image Processing API

Install packages
>>npm install

Prettier
>>npm run prettier

ESLint
>>npm run lint

Jasmine Testing
>>npm run test



Build the project
npm run build

Start the server
>>npm run start

after start the server got to http://localhost:3000/api/imageprocess

then write image name ex: pic,pic1,pic2,pic3
the output will be in image/output
